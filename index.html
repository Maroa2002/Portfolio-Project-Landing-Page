Features

Temperature Monitoring

Monitor and maintain optimal temperature levels in your home.

Light Intensity Detection

Automatically adjust lighting based on ambient light intensity.

Security with Ultrasonic Sensors

Ensure home security with advanced ultrasonic sensors.

Gas Detection

Detect and alert about potential gas leaks to ensure safety.

About

This project was inspired by the increasing need for smarter home solutions. Observing the potential of IoT technology to transform home environments, I set out to develop a system that demonstrates practical applications, enhancing everyday living through automation and smart monitoring.

This is a Portfolio Project for Holberton School.
